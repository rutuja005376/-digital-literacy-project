About the Project
We spend so much time online—for classes, assignments, chatting, and everything else—that it’s easy to forget how important it is to use the internet properly.
This project helped me explore simple but important things like staying safe online, building a good digital profile, communicating professionally, and learning basic technical skills. I’ve kept everything easy to understand and based on real-life situations.
